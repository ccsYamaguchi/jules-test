# テトリスゲーム

シンプルなテトリスゲームです。Webブラウザで動作します。

## 利用方法

1.  このリポジトリのファイルをダウンロードまたはクローンします。
2.  `index.html` ファイルをWebブラウザで開きます。
3.  「ゲーム開始」ボタンをクリックすると、ゲームが始まります。

## 操作説明

-   **← (左矢印キー)** または **Aキー**: ブロックを左に移動します。
-   **→ (右矢印キー)** または **Dキー**: ブロックを右に移動します。
-   **↑ (上矢印キー)** または **Wキー**: ブロックを回転させます。
-   **↓ (下矢印キー)** または **Sキー**: ブロックを1段下に移動します（ソフトドロップ）。
-   **スペースキー**: ブロックを一気に一番下まで落下させます（ハードドロップ）。

ゲームオーバーになるとスコアが表示されます。「ゲーム開始」ボタン（ゲームオーバー後は「リスタート」の表示になっているかもしれません）を押すと、新しいゲームを始めることができます。
