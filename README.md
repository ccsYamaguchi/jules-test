# テトリスゲーム (TypeScript版)

シンプルなテトリスゲームです。Webブラウザで動作します。このバージョンはTypeScriptで記述されています。

## プロジェクトのセットアップと実行

### 前提条件

- Node.js と npm (または yarn) がインストールされていること。

### セットアップ

1.  このリポジトリのファイルをダウンロードまたはクローンします。
    ```bash
    git clone <リポジトリURL>
    cd <リポジトリ名>
    ```
2.  プロジェクトの依存関係をインストールします。
    ```bash
    npm install
    ```
    または
    ```bash
    yarn install
    ```

### ビルド

以下のコマンドで、TypeScriptソースコードをJavaScriptにコンパイル・バンドルします。成果物は `dist` ディレクトリに出力されます。
```bash
npm run build
```

### 開発サーバーでの実行

開発サーバーを起動し、ブラウザでゲームをリアルタイムに確認できます。ファイル変更時には自動的にリロードされます。
```bash
npm run start
```
通常、`http://localhost:9000` でアクセス可能になります。

### 静的ファイルの直接実行

ビルド後、`dist/index.html` ファイルを直接Webブラウザで開くことでもゲームを実行できます。

## 利用方法 (ビルド済みファイルの場合)

1.  ビルド済みの `dist` ディレクトリの内容を取得します。
2.  `dist/index.html` ファイルをWebブラウザで開きます。
3.  「ゲーム開始」ボタンをクリックすると、ゲームが始まります。

## 操作説明

-   **← (左矢印キー)** または **Aキー**: ブロックを左に移動します。
-   **→ (右矢印キー)** または **Dキー**: ブロックを右に移動します。
-   **↑ (上矢印キー)** または **Wキー**: ブロックを回転させます。
-   **↓ (下矢印キー)** または **Sキー**: ブロックを1段下に移動します（ソフトドロップ）。
-   **スペースキー**: ブロックを一気に一番下まで落下させます（ハードドロップ）。
-   **Cキー** または **Shiftキー**: ブロックをホールドします。

ゲームオーバーになるとスコアが表示されます。「ゲーム開始」ボタンを押すと、新しいゲームを始めることができます。
